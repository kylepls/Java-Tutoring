#### Assignment Operator

    variable = 1

Add and assign

    variable += 1

Example:

    int i = 0;
    i += 1;
    // i == 1
    
Is the same as:
    
    i = i + 1;
    // i == 1

    variable = variable + 1;
    
Subtract and assign

    variable -= 1;
    
Multiply and assign

    variable *= 2;
    
Divide and assign

    variable /= 2
    
Modulus and assign

    variable %= 2
